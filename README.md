# Sistema de Gestión de Vehículos

## Nombre del proyecto
Sistema de Gestión de Vehículos (CRUD)

---

## Descripción
Este proyecto es una aplicación web desarrollada para la gestión de vehículos, permitiendo realizar operaciones básicas como crear, listar, editar y eliminar registros.

También cuenta con un sistema de inicio de sesión para controlar el acceso a la información y un panel principal (dashboard) donde se administra todo el sistema.

El proyecto está dividido en frontend y backend, consumiendo una API REST y conectado a una base de datos MongoDB.

---

## Características principales

- Inicio de sesión de usuarios
- Panel principal (dashboard)
- Registro de vehículos
- Listado de vehículos
- Edición de vehículos
- Eliminación de vehículos
- Consumo de API REST
- Interfaz responsiva
